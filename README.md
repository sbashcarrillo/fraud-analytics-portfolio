# Fraud Analytics Portfolio

This project is a multi-page portfolio website built using HTML and CSS as part of WGU Web Development Foundations (D276).

## Overview

The site includes:
- Resume page highlighting experience in fraud prevention and analytics
- Cover letter outlining professional background and goals
- Career goals page with multimedia elements and a contact form

## Features

- Semantic HTML structure (header, nav, main, section, article, footer)
- Custom CSS styling with responsive layout elements
- Embedded media (image, audio, video)
- Functional contact form
- Multi-page navigation

## Technologies Used

- HTML5
- CSS3

## Purpose

This project demonstrates foundational web development skills and serves as a starting point for a broader data and analytics portfolio.
